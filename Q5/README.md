Title

Program to Find the Maximum Element in an Array (Least Time Complexity)

Aim

To find the largest element in an array using the minimum possible time complexity.

Algorithm
Read the array elements.
Assume the first element is the maximum.
Traverse the remaining elements.
If an element is greater than the current maximum, update it.
Print the maximum element.

Time Complexity

O(n)

(Each element is visited exactly once, which is the least possible time complexity for this problem.)

Space Complexity

O(1)

(Only one extra variable is used.)

Result

The program successfully finds the maximum element in an array using linear time O(n) and constant space O(1), producing the expected outputs for all given test cases.
