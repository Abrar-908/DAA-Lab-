Title

Bubble Sort of an Integer Array

Aim

To sort an array of integers in ascending order using the Bubble Sort technique.

Algorithm
Read the array elements.
Compare adjacent elements.
Swap them if they are in the wrong order.
Repeat the process for all elements.
Continue until the array becomes sorted.

Time Complexity (Big-O Notation)
Case	Complexity
Best Case (already sorted, optimized version)	O(n)
Average Case	O(n²)
Worst Case	O(n²)

For the above implementation (without optimization), the overall time complexity is:

O(n²)

Space Complexity

O(1)

(Bubble Sort performs sorting in-place without using extra memory.)

Result

The program successfully sorts the given array using the Bubble Sort algorithm. The algorithm has a time complexity of O(n²) and a space complexity of O(1).
