Title

Merge Sort for Sorting an Integer Array

Aim

To sort an array in ascending order without using built-in sorting functions and achieve O(n log n) time complexity.

Algorithm
Divide the array into two halves.
Recursively sort both halves.
Merge the sorted halves into one sorted array.
Repeat until the entire array is sorted.

Time Complexity

O(n log n)

Space Complexity

O(n)

Result

The program successfully sorts the array using Merge Sort without built-in functions, achieving O(n log n) time complexity with O(n) auxiliary space.
